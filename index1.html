About

        This project is an automated irrigation system that integrates weather data for remote and local control. It is a customizable solution designed for medium and large scale agriculture. The system allows for efficient and optimized irrigation based on current weather conditions. Its automated features help to save time, money and resources, making it an ideal solution for farmers and other agricultural applications.
